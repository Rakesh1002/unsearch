## Your First Search

Let's perform a simple web search:

<CodeGroup>

```bash cURL
curl -X POST "https://api.unsearch.dev/api/v1/search" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "artificial intelligence trends 2024",
    "max_results": 5
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.unsearch.dev/api/v1/search",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "query": "artificial intelligence trends 2024",
        "max_results": 5
    }
)

results = response.json()
print(results)
```

```javascript Node.js
const axios = require('axios');

const response = await axios.post(
  'https://api.unsearch.dev/api/v1/search',
  {
    query: 'artificial intelligence trends 2024',
    max_results: 5
  },
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  }
);

console.log(response.data);
```

</CodeGroup>

## Search with Scraping

To get the actual content from search results:

<CodeGroup>

```bash cURL
curl -X POST "https://api.unsearch.dev/api/v1/search" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "machine learning tutorials",
    "scrape_results": true,
    "max_results": 3,
    "scrape_options": {
      "extract_images": true,
      "extract_metadata": true
    }
  }'
```

```python Python
response = requests.post(
    "https://api.unsearch.dev/api/v1/search",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "query": "machine learning tutorials",
        "scrape_results": True,
        "max_results": 3,
        "scrape_options": {
            "extract_images": True,
            "extract_metadata": True
        }
    }
)
```

```javascript Node.js
const response = await axios.post(
  'https://api.unsearch.dev/api/v1/search',
  {
    query: 'machine learning tutorials',
    scrape_results: true,
    max_results: 3,
    scrape_options: {
      extract_images: true,
      extract_metadata: true
    }
  },
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
```

</CodeGroup>

## Common Use Cases

### 1. Market Research
```python
# Search for competitor information
results = client.search(
    "SaaS pricing strategies 2024",
    engines=["google", "bing"],
    scrape_results=True
)
```

### 2. Content Aggregation
```python
# Aggregate news from multiple sources
news = client.search(
    "technology news today",
    time_range="day",
    max_results=20
)
```

### 3. SEO Analysis
```python
# Check search rankings
rankings = client.search(
    "site:yoursite.com your keywords",
    engines=["google"],
    max_results=100
)
```
