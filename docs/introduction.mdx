## Welcome to UnSearch

UnSearch is the **verifiable retrieval primitive** for AI agents. Every search result returns a signed citation envelope, a content-addressable snapshot, and a `verify_claim` endpoint that grades any claim against any source with span-level evidence. Built for legal, medical, finance, and other regulated AI teams who can't ship native LLM search past their compliance review.

Apache 2.0. MCP-native. Self-hostable on Cloudflare.

## The problem we solve

AI agents confidently make claims tied to web sources that are wrong, dead, paraphrased, fabricated, or silently changed since the agent retrieved them. There is no widely-adopted infrastructure primitive that produces a cryptographically verifiable record of "what the agent saw, at what URL, at what time, with what hash" — so every team building agents for regulated workflows is rebuilding this primitive from scratch.

The numbers as of Q1 2026:

- **$145K in U.S. court sanctions in Q1 2026 alone** for AI-hallucinated legal citations
- **Harvey AI ($8B valuation) still hallucinates 1-in-6 queries**
- **40–60% medical RAG reference fabrication** without retrieval grounding
- **EU AI Act Article 12 full enforcement begins August 2026** — provenance documentation required; €15M / 3% turnover penalty

## For whom

UnSearch is built for:

- **Engineering leads at regulated-AI startups** (Series Seed–B) building vertical AI for legal, medical, finance, insurance, research, or compliance verticals.
- **AI platform directors at established regulated companies** (banks, hospital systems, insurance carriers, BigLaw firms, pharma) retrofitting LLM features ahead of EU AI Act enforcement.
- **Citation-integrity research and journalism engineers** snapshotting cited sources so retraction-tracking and reproducibility survive bit rot.

UnSearch is **not** built for indie devs building non-regulated agents — Anthropic's native `web_search` and OpenAI Codex CLI search are free and good enough for them.

## Get started

The fastest path — no signup, no API key, Free tier built in:

```bash
claude mcp add unsearch
# or, in any MCP-compatible client:
npx @unsearch/mcp-server
```

Then ask your agent to search and verify a claim. For SDK and REST quickstarts, see [Quickstart](/quickstart).

```python
from unsearch import UnSearch

client = UnSearch(api_key="uns_...")
hits = client.search({"query": "EU AI Act Article 12 logging requirements"})
verify = client.verify_claim({
    "claim": "EU AI Act full enforcement begins August 2026",
    "source_url": hits["results"][0]["url"],
})
print(verify["supported"], verify["confidence"], verify["evidence_spans"])
```

```ts
import { UnSearch } from "@unsearch/sdk"

const client = new UnSearch({ apiKey: process.env.UNSEARCH_API_KEY! })
const hits = await client.search({ query: "EU AI Act Article 12 logging" })
const verify = await client.verifyClaim({
  claim: "EU AI Act full enforcement begins August 2026",
  source_url: hits.results[0].url,
})
```
