## Welcome to UnSearch API

UnSearch is a powerful, privacy-focused web search and content scraping API that gives you complete control over your search and data extraction needs without compromising user privacy.

## Core Features

- **🔍 Unified Search API**: Single endpoint for searching across multiple engines
- **📊 Content Extraction**: Automatic extraction of text, metadata, and structured data
- **⚡ High Performance**: Async architecture with Redis caching and connection pooling
- **🔐 Secure**: JWT authentication, API key management, and rate limiting
- **💳 Flexible Billing**: Free tier with generous limits, Pro plan for unlimited usage
- **🔄 Batch Processing**: Process multiple searches or scrapes in parallel
- **🪝 Webhooks**: Real-time notifications for async operations
- **📈 Analytics**: Detailed usage tracking and monitoring

## Quick Example

```bash
curl -X POST "https://api.unsearch.dev/api/v1/search" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "artificial intelligence",
    "engines": ["google", "bing"],
    "scrape_results": true,
    "max_results": 10
  }'
```

## SDKs & Libraries

<CodeGroup>

```python Python
pip install unsearch-python

from unsearch import UnSearchClient

client = UnSearchClient(api_key="YOUR_API_KEY")
results = client.search("artificial intelligence")
```

```javascript Node.js
npm install unsearch-js

import { UnSearchClient } from 'unsearch-js';

const client = new UnSearchClient({ apiKey: 'YOUR_API_KEY' });
const results = await client.search('artificial intelligence');
```

```go Go
go get github.com/unsearch/unsearch-go

import "github.com/unsearch/unsearch-go"

client := unsearch.NewClient("YOUR_API_KEY")
results, _ := client.Search("artificial intelligence")
```

</CodeGroup>
